Die Entstehungsgeschichte von "Hello World!"

Die Verwendung von "Hello World!" als Ausdruck in der Programmierung geht auf das Jahr 1974 zurück, als Brian Kernighan, ein Informatiker bei Bell Labs, ein Buch mit dem Titel "The C Programming Language" schrieb. In diesem Buch wird "Hello, World!" als Beispielprogramm in der Programmiersprache C vorgestellt. Das Programm zeigt eine einfache Textausgabe, die den Ausdruck "Hello, World!" enthält.

Kernighan selbst sagte später, dass er den Ausdruck "Hello, World!" aus einem Comic Strip entnommen hatte, in dem eine Figur einen Computer bedient und den Ausdruck als Test für den Drucker verwendet. Das Buch von Kernighan und Ritchie wurde zu einem Bestseller und trug wesentlich zur Popularität von C bei.
